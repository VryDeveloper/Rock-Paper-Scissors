## About

A Rock Paper Scissors game with a Minecraft-inspired theme. The game combines the classic Rock Paper Scissors mechanics with clicker game elements — players can complete missions and use an in-game shop to enhance their experience.

---

## Technologies

- **HTML** — Game structure
- **CSS** — Styling and animations
- **JavaScript** — Game logic and interactivity

---

## Features

- **Gameplay** — Play against the AI in classic Rock Paper Scissors
- **Mission System** — Complete challenges to earn rewards
- **Shop** — Purchase upgrades and exclusive items
- **Themed Design** — Minecraft-inspired visuals

---

## How to Play

1. Choose between Rock, Paper or Scissors.
2. The game determines the winner based on classic rules.
3. Complete missions to earn points.
4. Spend your points on items in the shop.

## Sobre o Projeto

Este projeto é um jogo de Pedra, Papel e Tesoura com uma temática inspirada no Minecraft. O jogo combina elementos clássicos de Pedra, Papel e Tesoura com mecânicas de clicker game! permitindo que os jogadores completem missões e utilizem uma lojinha para melhorar sua experiência.

## Tecnologias Utilizadas

- **HTML**: Estrutura do jogo
- **CSS**: Estilização e animações
- **JavaScript**: Lógica do jogo e interatividade

## Funcionalidades

- **Modo de Jogo**: Jogue contra a IA no tradicional Pedra, Papel e Tesoura.
- **Sistema de Missões**: Complete desafios para ganhar recompensas.
- **Lojinha**: Adquira melhorias e itens exclusivos.
- **Design Temático**: Visual inspirado no Minecraft.

## Como Jogar

1. Escolha entre Pedra, Papel ou Tesoura.
2. O jogo determinará o vencedor com base nas regras clássicas.
3. Complete missões para ganhar pontos.
4. Utilize seus pontos para comprar itens na loja.
